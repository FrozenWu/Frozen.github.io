Markdown基本语法

发表于 2019-08-27 | 更新于: 2019-08-27 | 分类于 博客

Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。本人使用的是Typora作为.md格式文件的编辑器，可以去Typora官网进行下载。

基本语法

标题

在想要设置为标题的文字前面加#来表示,#是一级标题,##是二级标题，以此类推。支持六级标题。在Typora中#后面需要跟一个空格。

字体

引用

在引用的文字前加>即可。引用也可以嵌套，如加两个>>三个>>>

分割线

三个或者三个以上的 - 或者 * 都可以。
